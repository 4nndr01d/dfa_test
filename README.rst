Базовые команды
--------------


Настройка окружения
^^^^^^^^^^^^^^^^^^^^^

* Создать файл .env(пример .env.sample)

Docker
^^^^^^^^^^^^^^^^^^^^^

* Запустить контейнер::

    $ sudo docker compose --env-file .env  up -d

Tests
^^^^^^^^^^^^^^^^^^^^^

* Запустить тесты::

    $ python manage.py tests

API
^^^^^^^^^^^^^^^^^^^^^

* Проект имеет OpenApi по урлу /docs/
